Multi-Tenant CMS
Architecture.

A unified monorepo powering multiple sites from a single codebase. Featuring a NestJS core and a dynamic, data-driven React frontend.

Architecture Stack

Public Site (React)

A high-performance React + Vite application. It acts as a headless CMS frontend, dynamically rendering pages and blocks based on JSON configurations fetched from the backend.

Core Backend (NestJS)

The central nervous system. Handles tenant resolution, serves static assets (CSS/Images) via proxy, and provides API endpoints for content retrieval.

Admin Panel (Angular)

The legacy frontend, now repurposed as a dedicated back-office interface for content managers to edit JSON files and site settings.
